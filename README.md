## 📖 What this is

A research-backed knowledge base on the AI-powered future of work — organized across **6 domains**, **148 directories**, and **400+ markdown articles**, with a handful of interactive HTML assessment tools. The content spans human-AI collaboration, workforce skills, organizational change, intelligent operations, metrics, and global trends.

It's also a working example of the thing it documents: an AI-assisted research and publishing workflow, built solo in days rather than months.

## 🤖 How it was built

- **[RivalSearchMCP](https://github.com/tatianathevisionary/RivalSearchMCP)** — multi-source research across academic papers, industry reports, and case studies to synthesize the knowledge frameworks.
- **Cursor** — built the site infrastructure: layouts, navigation, and index pages.
- **GitHub Pages** — Jekyll-based deployment with GitHub Actions.

The project itself is the argument: AI tools amplifying a single author's research and synthesis — the future of work, demonstrated rather than described.
